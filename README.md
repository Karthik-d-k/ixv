# ixv

:crab: A CLI application for verifying intel hex file.


## Usage

```bash
$ ixv [OPTIONS] <HEX_FILE>
```

**Arguments:**

```
<HEX_FILE>  Path to an hex file
```

**Options:**

```
  -p, --pb       Show progress bar
  -h, --help     Print help
  -V, --version  Print version
```


## About tool name

> `ixv` ==> **i**ntel he**x** **v**erifier
